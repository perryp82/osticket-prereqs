<h2>Installation Steps</h2>
<h3>1.) Creating a Virtual Machine</h3>
<p>
<img src="https://imgur.com/a/EkdbY82" height="80%" width="80%" alt="Disk Sanitization Steps"/>
</p>
In Microsoft Azure, we will create a VM that we'll add to a new Resource Group, titled "osTicket". For the VM name, we will title it "osticket-vm". Our VM will use the image "Windows 10 Pro, version 22H2 - x64 Gen2". As well as this, the size of our VM should include at least 2vcpus, and 8GiB of memory. Then we will create a username & password to access our VM, as well as check the licensing box at the bottom. Lastly, we will review & create our VM. We do not need to make any adjustment to any management, disks, or networking sections.
